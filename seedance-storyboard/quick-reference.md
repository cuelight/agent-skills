# Seedance Storyboard 快速参考

## 默认工作方式

- 先出一版完整 `Prompt Draft`
- 只在缺少关键事实时追问 1-3 个问题
- 默认用 `镜头1 / 镜头2 / ...`，不强制写每个镜头精确秒数
- 默认不写 BGM、配乐、背景音乐、音乐氛围
- prompt 正文不写 `【约束条件】`

## 镜头数量

| 总时长 | 推荐镜头数 | 说明 |
| --- | --- | --- |
| 4-6 秒 | 1-3 镜 | 一镜到底、单一动作、单一情绪转折 |
| 7-10 秒 | 2-5 镜 | 建立 -> 变化 -> 收束 |
| 11-15 秒 | 3-8 镜 | 小型叙事弧线，多人物或多动作阶段 |

> 1-8 镜头是范围，不是硬凑。复杂度服从动作、情绪和模型稳定性。

## 单镜头四要素

每个 `镜头N` 按固定顺序写：

```text
景别+运镜/切换 -> 主体动作与表情 -> 位置/空间变化 -> 同步声音
```

标准句式：

```text
镜头1：medium close-up（中近景） static shot（固定镜头），角色右手缓慢攥紧衣角、眼神从躲闪变得坚定，窗边冷光落在她肩上，手机震动声短促响起。
```

## 画幅景别表

| 画幅 | 优先景别 | 不建议滥用 | 适合场景 |
| --- | --- | --- | --- |
| 16:9 横屏 | `wide shot` / `medium shot` / `two shot` / `over-the-shoulder` / `medium close-up` | 只用近景讲空间戏 | 空间建立、多人关系、横向调度、广告、动作 |
| 9:16 竖屏 | `medium shot` / `medium close-up` / `close-up` / 紧凑 `two shot` | 横向群像、长距离远景 | 单人情绪、社媒短内容、近距离关系 |

竖屏里的 `wide shot` 只用于必要空间建立；`extreme close-up` 只用于证据物或强情绪落点。

## 景别层级与跨景别

景别层级：

```text
wide shot -> medium shot -> medium close-up -> close-up -> extreme close-up
```

- 相邻镜头默认避免无动机只差一级景别
- 优先跨一个以上景别，或让切换承担新信息
- 相邻景别可以保留，但必须写清剪辑动机：反应、证据、动作阶段、情绪升级

## 跨分镜组衔接

多段 prompt 或多 item 输出时，检查上一段最后镜头到下一段首镜头。

可用衔接方式：

- 空间重建：新场景先用建立镜头
- 动作接续：承接上一个动作结果
- 声音桥：门响、脚步、手机震动、人群声延续
- 情绪递进：上一段压住，下一段爆发或收束
- 道具接力：上一段出现的物件成为下一段信息中心

不要只写“承接上一段氛围”。要写具体桥，例如“上一段纸张摩擦声延续，下一段镜头切到她把文件放在桌上”。

## 最稳镜头选择

| 目标 | 优先写法 |
| --- | --- |
| 建立空间 | `wide shot` / `establishing shot` |
| 稳定叙事 | `medium shot` |
| 双人关系 | `two shot` / `over-the-shoulder` |
| 听者反应 | `reaction shot` |
| 情绪推进 | `medium close-up` + `slow push in` |
| 情绪落点 | `close-up` |
| 关键证据 | `close-up` / 少量 `extreme close-up` |
| 冷静压抑 | `static shot` |
| 现场紧张 | 轻微 `handheld` |

## 人物站位速查

| 关系 / 情绪 | 站位策略 |
| --- | --- |
| 普通对话 | 一步距离 + 同一深度 + 同框 |
| 紧张对峙 | 数步之外 + 同一深度 + 保持同框 |
| 权力压迫 | 前景压人 + 后景孤立 |
| 暧昧 | 近距离同框 + 视线不直接对视 |
| 入场仪式 | 从画面深处缓步走来 |
| 关系撕裂 | 从同框切到单人反应 |
| 多人群像 | 主角前景 + 其他人后景分层 |

## 声音速查

| 类型 | 标记 | 用途 |
| --- | --- | --- |
| 普通对白 | `dialogue` / `{台词}` | 画内说话，嘴型同步 |
| 旁白 | `narration` | 推进剧情或说明时间跳跃 |
| 内心独白 | `internal_monologue` | 画内无嘴型心理声 |
| 画外音 | `voice_over` | 回忆、跨时空、电话外声 |
| 场外对白 | `off_screen` | 门外、画外、电话另一端 |
| 环境语音 | `ambient_voice` | 人群议论、广播、市场声 |
| 环境声 | environmental sound | 雨声、风声、车流、空调低鸣 |
| 动作音效 | action sound | 门响、脚步、纸张、手机震动 |

默认不写音乐。用户明确要求音乐时，才把音乐作为声音段的独立说明。

## Prompt 骨架

```text
[风格]，约[#]秒，[16:9/9:16]，[情绪或质感]

【分镜时序】
镜头1：wide shot（远景） static shot（固定镜头），...，...，...
镜头2：medium close-up（中近景） slow push in（缓慢推近），...，...，...

【声音】
环境声、动作音效、必要对白或旁白；默认不写音乐。

【参考素材】
@图片1 作为角色形象参考，@视频1 参考动作节奏，@音频1 作为对白音色或环境声参考
```
