# Seedance Storyboard 快速参考

## 默认工作方式

- 默认先出一版完整 prompt 首稿
- 只有缺少关键事实时才追问 1-3 个问题
- 缺少信息时优先补：时长、主体动作、素材情况、画幅

## 最稳的镜头选择

| 目标 | 优先写法 |
| --- | --- |
| 建立空间 | `wide shot` |
| 稳定叙事 | `medium shot` |
| 情绪落点 | `close-up` |
| 对白关系 | `two shot` / `over-the-shoulder` / 正反打 |
| 情绪升温 | `slow push in` |
| 冷静压抑 | `static shot` |
| 现场紧张 | `handheld`（轻微手持感） |

## 人物关系镜头速查

| 场景 | 推荐写法 |
| --- | --- |
| 常规双人对话 | `two shot` + `外反打` |
| 逼问 / 争吵 / 告白 | `外反打` 建立后切 `内反打` |
| 听的人更重要 | `reaction shot` |
| 亲密 / 对立同框 | `two shot` |
| 权力压迫 | 前景压人 + `close-up` / `low angle` |

## 风格流派速查

| 流派 | 适用场景 | 关键词 |
| --- | --- | --- |
| 古典叙事 | 剧情片、短剧主线、关系戏 | 清楚、稳定、叙事优先 |
| 精准控制 | 悬疑、压迫、制度空间 | 对称、静稳、克制慢推 |
| 纪实手持 | 冲突、危机、追逐 | 近距离、轻手持、现场感 |
| 长镜头调度 | 漫游、仪式、连续动作 | `long take`、连续走位 |
| 情绪化碎片 | 都市夜景、暧昧、孤独 | 遮挡、霓虹、停顿、反射 |
| 类型化强风格 | 犯罪、黑色幽默、强包装 | 夸张构图、强节奏、作者感 |

## 短时长约束

- 4-6 秒：1-2 个主要信息点
- 7-10 秒：2-3 个节拍
- 11-15 秒：可做小型叙事弧线，但仍要克制切镜

避免：

- 过多切镜
- 多个复杂运镜同时出现
- 复杂动作 + 密集对白 + 大量特效同时叠加

## Prompt 结构

```text
[整体风格 + 时长 + 比例 + 基调]

0-X秒：环境 / 景别或机位 / 运镜 / 主体动作 / 情绪或声音
X-Y秒：环境 / 景别或机位 / 运镜 / 主体动作 / 情绪或声音

【声音】...
【参考素材】@图片1 ...，@视频1 ...
```

## 多模态引用速查

```text
@图片1 作为角色形象参考
@图片2 作为首帧
@视频1 参考运镜和动作节奏
@音频1 用于配乐氛围参考
```

## 典型默认值

- 叙事场景：优先 `medium shot`
- 对话场景：优先 `two shot` + `外反打`
- 情绪场景：优先 `close-up` + `slow push in`
- 广告场景：优先 `static shot` + `close-up`
- 动作场景：优先 `tracking shot`，少量切镜
