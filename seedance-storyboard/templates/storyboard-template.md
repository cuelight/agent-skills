# Seedance 分镜模板

以下模板不是死板套表，而是帮助 agent 快速选择适合当前场景的镜头策略。

## 模板一：叙事对白场景

适用：双人关系戏、问答、试探、争执、告白

```text
[整体风格]，[#秒]，[#比例]，[空间氛围]

0-X秒：`two shot` 或 `wide shot` 建立两人关系与站位，[角色A] 与 [角色B] 在 [场景] 中形成 [亲密/对立/试探] 状态
X-Y秒：`medium close-up` 平视 `over-the-shoulder` 固定镜头，聚焦 [角色A] 的表达，[动作/台词]，眼神和呼吸暴露 [情绪]，[环境物件/光线] 出现轻微变化
Y-Z秒：切到 [角色B] 的 `reaction shot` 或 `内反打`，低机位或平视机位轻微推近，[动作/表情/短对白]，背景声压低关系张力
Z-结束：`close-up` 平视固定镜头收在关键心理落点或关系变化，保留 [环境音/动作余音] 作为情绪尾韵

【声音】对白语气 + 环境音 + 配乐基调
【参考素材】@图片1 ...，@视频1 ...
```

## 模板二：情绪特写场景

适用：崩溃、犹豫、觉醒、发现异常、压抑爆发

```text
[整体风格]，[#秒]，[#比例]，[情绪基调]

0-X秒：[环境锚点]，`medium shot` 或 `static shot`，角色处在 [空间] 中，动作克制，情绪被压住
X-Y秒：`medium close-up` 平视 `slow push in` 逼近角色面部或手部，捕捉 [眼神/呼吸/肩背/指尖] 的情绪表演，[光线/雨影/尘土/热气] 在背景中变化
Y-结束：`close-up` 或 `extreme close-up` 固定镜头落到关键表情、眼泪、指尖、物件或发现对象，声音收成 [环境音/轻微喘息/低频氛围]

【声音】环境安静 / 低频氛围 / 单一关键音效
【参考素材】@图片1 ...，@音频1 ...
```

## 模板三：动作场面

适用：打斗、追逐、舞蹈、快速移动、能量释放

```text
[整体风格]，[#秒]，[#比例]，[动作基调]

0-X秒：`wide shot` 或 `medium wide shot` 高/低机位建立动作空间与角色准备状态，环境物件被风、尘土或人群扰动
X-Y秒：`medium shot` 平视或低机位 `tracking shot` 展示核心动作起势，主体完成 [冲刺/格挡/闪避]，表情紧绷，动作声清晰
Y-Z秒：跟随主体动作推进，强调动势、碰撞、节奏，不堆过多复杂运镜；背景光线、烟尘或碎屑随动作变化
Z-结束：用 `close-up`、定格姿态或 `pull out` 收束动作结果，保留 [撞击声/喘息/环境回声]

【声音】动作音效 + 节奏型配乐
【参考素材】@图片1 角色参考，@视频1 动作节奏参考
```

## 模板四：广告展示场景

适用：产品、品牌、材质、食物、美妆、珠宝、科技产品

```text
[广告风格]，[#秒]，[#比例]，[光影与质感]

0-X秒：`close-up` 或微距镜头，抓产品最强视觉卖点
X-Y秒：`static shot` 或 `slow push in`，展示主体轮廓、材质和结构
Y-Z秒：切到使用场景或功能演示，画面保持整洁，动作明确
Z-结束：`close-up` 或稳定中景完成品牌落点 / slogan / 包装定格

【声音】音乐风格 + 关键使用音效
【参考素材】@图片1 产品外观，@图片2 材质参考，@视频1 节奏参考
```

## 模板五：都市情绪 / 夜景氛围场景

适用：孤独、暧昧、擦肩而过、回忆、城市夜晚

```text
[整体风格]，[#秒]，[#比例]，[都市夜景/霓虹/雨夜等氛围]

0-X秒：通过前景遮挡、反射或霓虹建立环境，角色处在空间边缘
X-Y秒：`medium shot` 或 `close-up`，角色动作很少，重点是停顿、眼神、呼吸和未说出口的情绪
Y-结束：保留情绪悬置，可用 `slow motion`、反射或 `close-up` 收束

【声音】环境音 + 低存在感配乐
【参考素材】@图片1 角色参考，@图片2 场景参考
```

## 模板六：视频延长 / 续拍

适用：接着拍、补剧情、延长动作、延长情绪尾韵

```text
将@视频1延长 [#秒]

延续@视频1的主体、风格、节奏与空间逻辑：
0-X秒：[与原片直接衔接的新动作或新镜头]
X-Y秒：[推进或转折]
Y-结束：[收束结果]

【要求】与前段无缝衔接，保持主体一致、动作连贯、光影和情绪统一
【参考素材】@视频1 原视频，@图片1 如需补角色或场景参考
```

## 模板七：视频编辑 / 剧情改写

适用：换角色、改动作、改表情、颠覆原剧情

```text
基于@视频1进行编辑：

【保留】原视频的 [场景/运镜/节奏/主体位置]
【修改】[具体改动点]
【新增】[新动作 / 新情绪 / 新结果]

输出要求：
- 保持镜头逻辑连续
- 只在指定位置发生明显变化
- 新内容与原视频视觉风格统一
```

## 模板八：旁白驱动型短剧

适用：短剧爽点、人物回归、反转揭示、跨时空跳跃；用一句旁白替代昂贵镜头堆砌。

```text
[整体风格]，[#秒]，[#比例]，[空间氛围]

0-X秒：`wide shot` 或 `establishing shot` 建立空间，[环境/人群反应]
        Narration：[1 句话讲清时空跳跃或人物处境，3-15 字]
X-Y秒：`medium shot` 低机位 `slow-motion tracking shot` 跟随主体动作起势，肩背、眼神和步伐把气场与状态外化，场景光线或人群位置随入场变化
Y-结束：切到旁观者的 `reaction shot` 或主体的 `close-up`，平视固定镜头定格在情绪/关系变化的瞬间，环境音短暂让位给低频鼓点

【声音】低频鼓点配乐 / 大场景环境音；旁白男声沉稳低音
【参考素材】@图片1 角色形象参考，@图片2 场景空间参考
```
