# 对白 / 旁白工业化标准（AIGC 视频）

适用：Seedance / Sora / Vidu / Runway / Kling / Minimax 等文生视频模型。

---

## 一、声音层级总览

当前 AIGC 视频**不是"对白驱动"**，而是：

> **视觉 + 情绪 + 声音点缀驱动**

与传统影视剧本的根本差别：传统剧本里"导演读懂即可"，但模型必须被**显式**告知：

- 谁在说？
- 是否出现在画面内？
- 是否需要嘴型同步？
- 是否属于回忆 / 内心 / 旁白？

否则会出现：嘴型错乱、人物乱说话、镜头不匹配、情绪失焦、叙事混乱。

真正高级的短剧 prompt：**少说话、强情绪、强镜头、强旁白、强停顿**。

---

## 二、6 类声音分类（核心）

| 类型 | 英文标记 | 是否有嘴型 | 是否在镜头内 | 推荐用途 |
|---|---|---|---|---|
| 普通对白 | `dialogue` | 有 | 是 | 人物交流、冲突、情绪爆点 |
| 旁白 | `narration` | 无 | 不一定 | 推进剧情、爽剧节奏 |
| 内心独白 | `internal_monologue` | 无 | 是 | 心理戏、情绪强化 |
| 画外音 | `voice_over` | 无 | 否 | 回忆、跨场景、时空跳跃 |
| 现场画外音 | `off_screen` | 有 | 否 | 房间外说话、电话另一端 |
| 环境语音 | `ambient_voice` | 不重要 | 不重要 | 群众嘈杂、背景氛围 |

> 在 prompt 或 DSL 中**显式标注类型**，模型才能正确决定是否生成嘴型、是否把声源放到画外。

---

## 三、长度规范（非常重要）

| 类型 | 推荐长度 |
|---|---|
| 单句对白 | **3–12 字** |
| 内心独白 | 1–2 句 |
| 旁白 | 1 句 |
| 连续对白 | ≤ 3 轮 |
| 单镜头总语音 | **≤ 8 秒** |

超过这个范围时优先做的事：拆镜头、改成旁白 / 内心独白、把信息塞进视觉。

---

## 四、各类型用法与写法

每类的结构 = 用途 → 写法要点 → Seedance 分镜 prompt 示例。

> **角色引用约定**：在 prompt 中用 `@图片N 作为角色X形象参考` 的方式给角色编号，台词标注里的角色名（角色A / 角色B）与参考素材保持一一对应。

### 1. Dialogue（普通对白）

**用途**：冲突、情绪爆点、人物关系。
**写法要点**：单句 3–12 字；长句一定拆。复杂动作戏不要硬塞对白。

**Seedance 分镜示例**（雨夜对峙）：

```text
冷调写实情绪风格，8秒，16:9，雨夜屋顶，潮湿压抑

0-3秒：`two shot` 建立两人关系，雨水顺着角色A的脸滑下，角色B站在两步之外，背景霓虹模糊
3-5秒：切到 `外反打 over-the-shoulder`，中近景聚焦角色A，眼眶泛红
        Dialogue — 角色A：「你骗我？」
5-8秒：切到角色B的 `reaction shot`，沉默不语，眼神轻微闪躲

【声音】雨声 + 远处车流 + 极弱低频氛围，对白克制清晰
【参考素材】@图片1 作为角色A形象参考，@图片2 作为角色B形象参考，@图片3 雨夜屋顶氛围参考
```

**反例**（长句 = 嘴型炸裂）：

```
Dialogue — 角色A：「我一直以为你是真心爱我的，可直到今天我才发现，
你从头到尾都在骗我……」
```

---

### 2. Narration（旁白）

**用途**：短剧 / 爽文改编 / 网文节奏——可以替代大量昂贵镜头。
**写法要点**：1 句话讲清时空跳跃或人物处境，剩下交给画面。

**Seedance 分镜示例**（爽剧回归型）：

```text
高级商业感写实风格，10秒，16:9，奢华酒店大堂，冷暖光交织

0-3秒：`wide shot` 展示酒店大堂全景，所有人转头看向入口
        Narration：「三年后，他回来了。」
3-7秒：`slow-motion tracking shot` 跟随角色A从旋转门缓步入场，西装挺括，气场压人
7-10秒：切到员工的 `reaction shot`，几人下意识起身，眼神惊愕

【声音】低频鼓点配乐渐起 + 大堂环境音，旁白男声沉稳低音
【参考素材】@图片1 作为角色A形象参考，@图片2 酒店大堂场景参考
```

---

### 3. Internal Monologue（内心独白）

**用途**：AIGC 短剧最容易出高级感——不需要嘴型同步、强化情绪、不增加镜头复杂度。
**写法要点**：只表达情绪 / 欲望，**不要解释剧情**。

**Seedance 分镜示例**（复仇内心戏）：

```text
冷峻电影感风格，6秒，16:9，远观婚礼仪式，光影偏冷

0-2秒：`wide shot` 婚礼现场全景，角色A站在远处廊柱阴影里
2-4秒：`close-up` 角色A的侧脸，表情几乎不动，眼神锁定主桌
        Internal Monologue — 角色A：「他们都会还的。」
4-6秒：`slow push in` 继续逼近，瞳孔反射出婚礼灯光，停在情绪最重的一刻

【声音】极低频弦乐 + 礼堂远景人声混响，无对白
【参考素材】@图片1 作为角色A形象参考，@图片2 婚礼仪式空间参考
```

**反例**（解释剧情）：

```
Internal Monologue — 角色A：「因为他当年背叛了我，所以我现在必须复仇。」
```

---

### 4. Voice Over（画外音 / V.O.）

**用途**：回忆、时间跳跃、多时空叙事。

**与 Narration 的区别**：

| 类型 | 用途 |
|---|---|
| Narration | 讲故事、推进剧情 |
| Voice Over | 情绪 / 记忆，常跨时空 |

**Seedance 分镜示例**（童年回忆）：

```text
怀旧胶片质感风格，8秒，16:9，向日葵田，金色夕阳

0-3秒：`wide shot` 向日葵田全景，小孩从画面右侧奔向远处
3-6秒：`tracking shot` 跟随小孩穿过花海，逆光剪影
        Voice Over — 角色A：「妈妈说过，人会消失两次。」
6-8秒：`close-up` 小孩抬头望向天空，画面停在风吹动花的一刻

【声音】温暖钢琴主题 + 远处虫鸣 + 微风，画外音是成年女声、低柔
【参考素材】@图片1 作为童年角色A形象参考，@图片2 向日葵田场景参考
```

---

### 5. Off Screen Dialogue（O.S.）

**用途**：房间外声音、门外喊话、电话另一端、未出场角色。
**写法要点**：明确写"画外"，模型才不会给画内人物加嘴型。

**Seedance 分镜示例**：

```text
都市夜景写实风格，6秒，9:16，昏暗走廊

0-3秒：`medium shot` 角色A独自站在走廊尽头，手指攥紧，画面只有他
3-6秒：`slow push in` 镜头逼近他的侧脸，表情凝固
        Off Screen — 角色B（画外）：「饭好了。」

【声音】走廊环境混响 + 远处厨房轻微锅碗声，画外女声温和
【参考素材】@图片1 作为角色A形象参考，@音频1 作为画外角色B 音色参考
```

---

### 6. Ambient Voice（环境语音）

**用途**：群众嘈杂、市场议论、广播声、背景氛围。
**写法要点**：不写具体台词，描述"声音性质"。

```text
【声音】环境语音：人群低声议论，远处车流，偶尔笑声；不需要具体台词
```

## 五、不推荐的 3 种写法（反例集中区）

### ❌ 1. 超长对白

```
Dialogue:
"其实从三年前开始，我就已经知道你背叛了我……"
```

问题：嘴型炸裂、节奏拖垮、模型失控。

### ❌ 2. 多人同时长对话

```
A: ……
B: ……
C: ……
D: ……
```

当前模型很难稳定控制多角色嘴型同步。**最多 2 人，连续不超过 3 轮**。

### ❌ 3. 一边复杂动作一边长对白

```
边打斗边说长句
```

模型会优先保动作 → 嘴型直接崩坏。打斗场景请用 **内心独白 / 画外音** 替代。

## 六、工业级镜头结构（最稳）

真正适合短剧工业化的单镜头模板：

```
[视觉动作] + [一句对白] + [一句情绪旁白]
```

**最稳 Seedance 示例**（雪中等待）：

```text
冷调电影感风格，6秒，16:9，雪夜空旷站台，孤独

0-3秒：`wide shot` 雪中独自站立的角色A，雪花在路灯下飘落
        Dialogue — 角色A：「我等过你。」
3-6秒：`slow push in` 镜头逼近她侧脸，眼神空洞
        Narration：「但他没有回来。」

【声音】极弱钢琴 + 风雪环境音；旁白女声低柔
【参考素材】@图片1 作为角色A形象参考，@图片2 雪夜站台场景参考
```

**4 个高频组合速查**：

| 组合 | 用途 |
|---|---|
| 视觉动作 + Narration | 最稳定（爽剧 / 节奏） |
| 特写 + Internal Monologue | 情绪最强（心理戏） |
| 慢动作出场 + Narration | 爽感最强（回归 / 反转） |
| 空镜 + Voice Over | 电影感最强（回忆 / 跨时空） |

## 七、JSON / DSL 结构（agent / pipeline 用）

如果你在做 Agent、剧本生成器、Pipeline、MediaSpec，把对白标准化到下面的结构。`character` 字段值用**人类可读名**（角色A / 角色B / 教官 / 新兵），并配 `referenceImageRef` 指向 Seedance 的 `@图片N`，让台词与多模态参考保持一一对应。

```json
{
  "scene": {
    "visual": "雨夜屋顶，潮湿压抑",
    "references": {
      "characters": [
        { "name": "角色A", "referenceImageRef": "@图片1" },
        { "name": "角色B", "referenceImageRef": "@图片2" }
      ],
      "scene": { "referenceImageRef": "@图片3" }
    },
    "dialogue": [
      {
        "type": "dialogue",
        "character": "角色A",
        "text": "你骗我？"
      }
    ],
    "voice": [
      {
        "type": "internal_monologue",
        "character": "角色A",
        "text": "我早就知道这一天会来。"
      }
    ]
  }
}
```

**字段标准（与第二节分类表一一对应）**：

| 字段值 | 对应类型 | 是否必须有 character |
|---|---|---|
| `dialogue` | 普通对白 | 是 |
| `narration` | 旁白 | 否 |
| `internal_monologue` | 内心独白 | 是 |
| `voice_over` | 回忆画外音 | 通常有 |
| `off_screen` | 现场画外音 | 是（即使不在画内） |
| `ambient_voice` | 环境语音 | 否 |

## 八、与既有模板 / 示例的衔接

本文档与 skill 内其他文件的口径：

- `templates/storyboard-template.md` 「模板一：叙事对白场景」给出**镜头编排骨架**（two shot / 反打 / reaction shot），本文给出**对白本身的分类与长度规范**。
- `examples/example-prompts.md` 「示例一：对白关系戏」展示了完整的对白场景 prompt 写法，本文给每类声音单独提供范式，二者互补。
- Seedance 多模态引用统一遵循 `SKILL.md` 「多模态素材组织规则」：
  - 角色形象 → `@图片N 作为角色X形象参考`
  - 场景空间 → `@图片N 场景参考`
  - 配乐 / 音色 / 对白参考 → `@音频N`

对白场景 prompt 末尾的标准参考素材行：

```text
【参考素材】@图片1 作为角色A形象参考，@图片2 作为角色B形象参考，@图片3 场景参考，@音频1 作为对白音色参考
```

## 九、最终原则（一句话）

> AIGC 视频不是用对白讲故事的，是用**画面 + 情绪 + 一两句精准声音**讲故事的。
> 写不下去时，**先砍对白，再加视觉**。
