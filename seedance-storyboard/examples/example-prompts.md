# Seedance 分镜 Prompt 示例

以下示例统一采用“整体风格 -> 时间轴分镜 -> 声音 -> 参考素材”的结构，并尽量使用稳定、可执行的镜头语言。

## 示例一：对白关系戏

场景：深夜训练室里，新兵向教官请求最后一次机会。

```text
冷调写实剧情风格，10秒，16:9，深夜训练室的压抑氛围

0-3秒：`wide two shot` 平视固定镜头建立两人关系，空旷训练室只亮着冷白顶灯，教官站得笔直，新兵停在几步之外，肩背绷紧，灯管轻微闪烁，远处空调低鸣压住空气
3-6秒：切到 `medium close-up over-the-shoulder` 外反打，平视机位轻微推近新兵，他右手攥住训练服下摆，压住呼吸，冷光在地面反射轻晃
        Dialogue — 新兵：「再给我一次机会。」
6-8秒：切到教官的 `reaction shot`，中近景低机位固定镜头，面部几乎不动，只有眼神轻微收紧，身后器械阴影压到墙面
8-10秒：进入教官一侧的 `内反打 close-up`，低机位缓慢推近，他缓慢逼近半步，嘴角停住，鞋底声在空旷地面回响
        Dialogue — 教官：「你最好配得上这句话。」

【声音】低频紧张氛围配乐，远处空调低鸣，对白克制清晰
【参考素材】@图片1 作为新兵形象参考，@图片2 作为教官形象参考，@图片3 作为训练室空间参考
```

## 示例二：动作场面

场景：武侠竹林中两人短兵相接。

```text
冷色写实武侠风格，10秒，16:9，薄雾竹林，空气湿冷，杀气压低

0-2秒：`wide shot` 平视固定镜头展示竹林深处的对峙空间，两人相隔数步站定，肩背压低，竹叶被风吹得持续晃动，远处风声拉开杀气
2-5秒：`medium wide tracking shot` 低机位跟随黑衣刀客突然前冲，长刀从画面右侧斜切而入，对手眼神一紧后侧身闪避并抽剑格挡，薄雾被动作带散
5-8秒：中景平视跟拍保持动作清晰，连贯展示一轮贴身交锋，人物呼吸急促但动作利落，竹叶和衣摆被刀风卷起，金属撞击声干脆刺耳
8-10秒：`close-up` 平视固定镜头收在两件兵器交错僵持的一瞬，人物眼神互相顶住，前景竹叶掠过并短暂遮挡画面，碰撞余音在竹林里回荡

【声音】冷峻弦乐 + 风声 + 金属碰撞音效
【参考素材】@图片1 角色A造型参考，@图片2 角色B造型参考，@视频1 参考动作节奏
```

## 示例三：广告展示

场景：一支高端护肤精华广告。

```text
高端商业广告风格，12秒，16:9，干净克制的暖白色空间，质感高级细腻

0-3秒：微距 `close-up` 对准精华瓶口，透明液体沿玻璃缓慢滑落，反光干净克制
3-6秒：`static shot` 转为中近景，产品静置在石材台面中央，柔和漫射光突出瓶身轮廓与材质
6-9秒：`slow push in` 靠近滴管细节，手指轻轻拧开瓶盖，液滴悬停后落下，强调轻盈与纯净
9-12秒：稳定 `close-up` 收在品牌名与瓶身正面，背景保持虚化，画面定格

【声音】低存在感高级电子氛围配乐，玻璃与液滴细节音效清晰
【参考素材】@图片1 产品外观参考，@图片2 材质与包装细节参考
```

## 示例四：都市情绪戏

场景：雨夜便利店门口，女生隔着玻璃看见前任却没有进去。

```text
都市情绪电影感，8秒，9:16，雨夜霓虹与便利店白光交织，潮湿、克制、略带失落

0-2秒：前景带雨滴和路灯反射，`medium shot` 平视固定镜头展示女生停在便利店门外，玻璃门把她和室内隔开，她肩膀微微收紧，雨滴沿玻璃下滑，便利店门铃声很轻
2-5秒：`close-up` 平视过玻璃反射拍她的侧脸，镜头缓慢推近，店内暖光和街边霓虹叠在一起，她的眼神停住却没有动作，呼吸在玻璃上留下浅雾
5-8秒：近景平视 `slow push in`，她低头吸了一口气，手指从门把前缓慢收回，嘴角僵住，雨声和店内模糊人声把未说出口的情绪留在结尾

【声音】雨声、便利店轻微环境音、极弱的都市氛围配乐
【参考素材】@图片1 角色参考，@图片2 雨夜便利店场景参考
```

## 示例五：视频延长 / 续拍

场景：延长一段滑板少年在海边滑行的视频。

```text
将@视频1延长 8 秒

延续@视频1的金色海边治愈风格与少年滑行节奏：
0-3秒：镜头继续跟在少年身后滑行，夕阳把海面照得发亮，滑板轮压过带水痕的地面
3-5秒：`tracking shot` 从侧后方切近一点，他微微侧身避开迎面跑来的小孩，动作自然连贯
5-8秒：镜头轻轻拉开成 `medium wide shot`，他重新找回平衡后向海边加速滑去，海风把衣角吹起，画面带着余韵收束

【声音】延续原视频的海风、轮子摩擦地面声和轻快配乐
【参考素材】@视频1 原视频，如需补角色一致性可加@图片1 作为角色参考
```

## 示例六：导演感转译

场景：用户说“想要一点 Wong Kar-wai 式的都市暧昧感”。

```text
都市情绪化碎片风格，10秒，9:16，狭窄夜色街巷里潮湿霓虹轻微晕染，情绪暧昧又克制

0-3秒：通过前景路人和雨伞遮挡建立街巷空间，女生站在招牌下，男生从画面深处慢慢靠近
3-6秒：`medium shot` 保持两人没有真正碰到彼此，只在同一块霓虹反光里短暂停住，空气像被压慢
6-8秒：切到带反射的 `close-up`，女生侧脸与玻璃中的男生轮廓重叠，画面带一点轻微失焦和情绪停顿
8-10秒：慢慢收在她眼神的细微变化上，不把关系说死，只留下未完成的靠近感

【声音】城市夜晚环境音 + 极低存在感的迷离氛围配乐
【参考素材】@图片1 角色A参考，@图片2 角色B参考，@图片3 夜巷霓虹场景参考
```

> 注：用户输入"Wong Kar-wai 式都市暧昧感"，但 prompt 里没有出现导演名，而是转译为：前景遮挡 / 反射构图 / 霓虹晕染 / 慢镜停顿 / 情绪悬置。这是 SKILL.md 流派映射规则的标准做法——把"某导演感"拆成可执行的构图、节奏、机位、运镜与情绪目标。
>
> 画幅差异：本例按 9:16 竖屏拍——优先 `close-up` + `slow push in`，双人"近距离一前一后"。若改 16:9 横屏，会调整为：扩成街道全景 + 对称构图 + horizontal tracking + 双人同深度并排同框，节奏放慢但情绪强度会下降——按短剧 / 都市情绪类型默认仍建议 9:16。

## 示例七：内心独白心理戏

场景：女主站在前夫婚礼远处的廊柱阴影里，决心复仇。

```text
冷峻电影感风格，6秒，16:9，远观婚礼仪式，光影偏冷

0-2秒：`wide shot` 婚礼现场全景，女主站在远处廊柱阴影里，几乎隐入背景
2-4秒：`close-up` 女主侧脸，表情几乎不动，眼神锁定主桌
        Internal Monologue — 女主：「他们都会还的。」
4-6秒：`slow push in` 继续逼近，瞳孔反射出婚礼灯光，停在情绪最重的一刻

【声音】极低频弦乐 + 礼堂远景人声混响，无对白
【参考素材】@图片1 作为女主形象参考，@图片2 婚礼仪式空间参考
```
