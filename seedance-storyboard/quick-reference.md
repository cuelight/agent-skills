# Seedance Storyboard 快速参考

## 默认工作方式

- 默认先出一版完整 prompt 首稿
- 只有缺少关键事实时才追问 1-3 个问题
- 缺少信息时优先补：时长、主体动作、素材情况、画幅

## 最稳的镜头选择

| 目标 | 优先写法 |
| --- | --- |
| 建立空间 | `wide shot` |
| 稳定叙事 | `medium shot` |
| 情绪落点 | `close-up` |
| 对白关系 | `two shot` / `over-the-shoulder` / 正反打 |
| 情绪升温 | `slow push in` |
| 冷静压抑 | `static shot` |
| 现场紧张 | `handheld`（轻微手持感） |

## 子分镜七要素

每个时间段默认按这个顺序写成自然句：

```text
景别 + 机位 + 运镜 + 主体动作 + 情绪表演 + 环境变化 + 声音设计
```

| 要素 | 稳定写法 |
| --- | --- |
| 景别 | `wide shot` / `medium shot` / `medium close-up` / `close-up` |
| 机位 | 平视 / 低机位 / 高机位 / 过肩 / 主观视角 |
| 运镜 | `static shot` / `slow push in` / `pull out` / `tracking shot` / `pan` |
| 主体动作 | 走近、停步、抬手、推开、拿起、收回、转身 |
| 情绪表演 | 眼神收紧、呼吸压住、肩背绷紧、嘴角停住、手指发抖 |
| 环境变化 | 灯光闪烁、门板晃动、雨水加重、尘土扬起、热气散开 |
| 声音设计 | 对白、旁白、环境声、动作音效、低频氛围、短暂停顿 |

## 人物关系镜头速查

| 场景 | 推荐写法 |
| --- | --- |
| 常规双人对话 | `two shot` + `外反打` |
| 逼问 / 争吵 / 告白 | `外反打` 建立后切 `内反打` |
| 听的人更重要 | `reaction shot` |
| 亲密 / 对立同框 | `two shot` |
| 权力压迫 | 前景压人 + `close-up` / `low angle` |

## 站位策略速查

| 关系 / 情绪 | 站位策略 |
| --- | --- |
| 普通对话 | 一步距离 + 同一深度 + 同框 |
| 紧张对峙 | 数步之外 + 同一深度 + 同框 |
| 权力压迫 | 前景压人 + 后景孤立 |
| 暧昧 | 近距离同框 + 视线不对视 |
| 入场仪式 | 从画面深处走来，缓步前移 |
| 关系撕裂 | 切单人，避免再同框 |
| 多人群像 | 主角前景 + 其他人后景虚化 |

## 画幅决策速查

| 画幅 | 优先景别 | 优先运镜 | 双人站位 | 典型场景 |
| --- | --- | --- | --- | --- |
| 16:9 横屏 | wide / medium | horizontal pan / tracking | 同深度并排同框 | 动作 / 广告 / 群像 / 风景 |
| 9:16 竖屏 | medium / close-up | slow push in / tilt | 近距离一前一后 | 短剧 / 单人情绪 / 社媒 |
| 2.35:1 宽幅 | wide / medium | static / slow tracking | 远距离 + 留白 | 电影感叙事 / 史诗 |

## 风格流派速查

| 流派 | 适用场景 | 关键词 |
| --- | --- | --- |
| 古典叙事 | 剧情片、短剧主线、关系戏 | 清楚、稳定、叙事优先 |
| 精准控制 | 悬疑、压迫、制度空间 | 对称、静稳、克制慢推 |
| 纪实手持 | 冲突、危机、追逐 | 近距离、轻手持、现场感 |
| 长镜头调度 | 漫游、仪式、连续动作 | `long take`、连续走位 |
| 情绪化碎片 | 都市夜景、暧昧、孤独 | 遮挡、霓虹、停顿、反射 |
| 类型化强风格 | 犯罪、黑色幽默、强包装 | 夸张构图、强节奏、作者感 |

## 短时长约束

- 4-6 秒：1-2 个主要信息点
- 7-10 秒：2-3 个节拍
- 11-15 秒：可做小型叙事弧线，但仍要克制切镜

避免：

- 过多切镜
- 多个复杂运镜同时出现
- 复杂动作 + 密集对白 + 大量特效同时叠加

## Prompt 结构

```text
[整体风格 + 时长 + 比例 + 基调]

0-X秒：景别 / 机位 / 运镜 / 主体动作 / 情绪表演 / 环境变化 / 声音设计
X-Y秒：景别 / 机位 / 运镜 / 主体动作 / 情绪表演 / 环境变化 / 声音设计

【声音】...
【参考素材】@图片1 ...，@视频1 ...
```

## 对白类型速查

> 长度规范：单句 3–12 字 / 单镜头 ≤ 8 秒 / 连续对白 ≤ 3 轮（详见 references/dialogue-narration.md）

| 类型 | 标记 | 嘴型 | 画内 | 用途 |
| --- | --- | --- | --- | --- |
| 普通对白 | `dialogue` | 有 | 是 | 冲突、情绪爆点 |
| 旁白 | `narration` | 无 | 不限 | 推进剧情、爽剧节奏 |
| 内心独白 | `internal_monologue` | 无 | 是 | 心理戏 |
| 画外音 | `voice_over` | 无 | 否 | 回忆、跨时空 |
| 场外对话 | `off_screen` | 有 | 否 | 门外、电话 |
| 环境语音 | `ambient_voice` | — | — | 群众、氛围 |

## 多模态引用速查

```text
@图片1 作为角色形象参考
@图片2 作为首帧
@视频1 参考运镜和动作节奏
@音频1 用于配乐氛围参考
```

## 典型默认值

- 叙事场景：优先 `medium shot`
- 对话场景：优先 `two shot` + `外反打`
- 情绪场景：优先 `close-up` + `slow push in`
- 广告场景：优先 `static shot` + `close-up`
- 动作场景：优先 `tracking shot`，少量切镜
- 画幅默认：短剧 / 单人情绪 / 社媒 → 9:16；动作 / 广告 / 群像 / 风景 → 16:9
