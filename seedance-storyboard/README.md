# Seedance Storyboard Skill

将一句想法、一个剧情片段或一组参考素材，快速整理成适合 Seedance 2.0 使用的专业分镜 prompt。

## 这次重构后的定位

这个 skill 现在默认采用“先出首稿”的工作方式，而不是传统的长轮问卷式引导。

它会优先：

- 直接理解用户已有信息
- 自动选择更合适的景别、机位、运镜和人物关系镜头
- 先给出一版完整可用的 prompt
- 只在缺少关键事实时补问极少量问题

适用场景：

- “帮我把这个想法写成 Seedance prompt”
- “参考这张图和这段视频写一个即梦分镜”
- “我要一段有电影感的双人对话戏”
- “我想要偏 Kubrick / Wong Kar-wai / 纪实手持感，但要能真正落到镜头语言”

## Skill 能做什么

- 把模糊创意扩成完整时间轴分镜
- 用稳定的镜头语言组织 prompt，而不是只堆风格词
- 处理人物关系镜头，例如 `two shot`、正反打、`内反打/外反打`
- 处理导演流派映射，把“某导演感”翻译成构图、节奏和运镜特征
- 支持多模态素材引用、视频延长、编辑、动作复刻、情绪戏和广告片段

## 使用预期

默认输出通常长这样：

1. 一版完整 `Prompt Draft`
2. 一小段 `Optional Tweaks`

只有在这些关键信息缺失时，才会先追问：

- 时长
- 主体是谁、在做什么
- 有没有参考素材
- 画幅比例

不会默认再问一大串“风格是什么、镜头怎么走、配乐要不要、色调冷暖”等问题。

## 一个典型例子

### 原始想法

> 一个女孩在便利店外面看见前任，想进去又没进去，想要一点都市电影感

### 优化后的 Prompt

```text
都市情绪电影感，8秒，9:16，雨夜便利店外，潮湿霓虹与室内白光叠在一起，克制、暧昧、略带失落

0-2秒：前景带雨滴和反光的 `medium shot` 建立便利店门口空间，女孩停在玻璃门外，脚步刚刚停住
2-5秒：切到带反射的 `close-up`，她隔着玻璃看见前任坐在店内，神情微微绷紧却没有说话
5-8秒：镜头 `slow push in`，她指尖靠近门把又慢慢收回，情绪停在没有进去的那一刻

【声音】雨声、便利店轻微环境音、低存在感的都市氛围配乐
【参考素材】@图片1 作为角色参考，@图片2 作为便利店夜景氛围参考
```

### 再举一个导演感例子

用户说：

> 想拍一个办公室审讯场景，要有一点 Fincher / Kubrick 那种压迫控制感

skill 不应只复述导演名，而应翻译成：

- 对称构图
- 稳定机位
- 冷色空间
- 克制的 `slow push in`
- 精准关系镜头和压迫式距离

## 文件说明

- `SKILL.md`：主技能逻辑，定义默认工作方式与镜头决策规则
- `quick-reference.md`：最常用镜头语法、关系镜头、流派速查
- `templates/storyboard-template.md`：按场景类型组织的 prompt 模板
- `examples/example-prompts.md`：覆盖对白、动作、广告、都市情绪、延长等场景的完整示例

## 设计原则

- 镜头语言优先于空泛形容词
- 叙事清晰优先于花哨炫技
- 短时长里控制镜头复杂度
- 优先输出可执行 prompt，而不是冗长解释
- “导演感”优先翻译成具体镜头策略，而不是简单模仿名字

## 备注

这是一个通用外部 skill，不依赖 `ai-drama` 内部数据库、CLI 或项目专属实现。
它只负责把输入组织成更专业、更稳定的 Seedance 分镜 prompt。
