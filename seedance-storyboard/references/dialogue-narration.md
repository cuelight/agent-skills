# 对白 / 旁白 / 同步声音标准

适用：Seedance / 即梦等 AIGC 视频 prompt。

核心原则：视频不是靠长对白讲清一切，而是靠画面、动作、情绪和一两句精准声音共同推进。

默认不写 BGM、配乐、背景音乐、音乐氛围。音乐属于后期或用户明确要求项。若用户明确要求音乐，可以单独写“用户要求的音乐：...”，不要把音乐当作默认声音设计。

## 声音必须绑定镜头

声音不要只放在 prompt 末尾泛泛总结。每个关键 `镜头N` 至少给一个可听见的锚点：

- 对白
- 旁白
- 内心独白
- 环境声
- 动作音效
- 短暂停顿或声音变清楚

声音要和画面动作、空间、情绪绑定，例如“纸张折叠声”“门轴轻响”“手机震动”“远处车流声变清楚”。

## 6 类人声分类

| 类型 | 英文标记 | 是否有嘴型 | 是否在镜头内 | 推荐用途 |
| --- | --- | --- | --- | --- |
| 普通对白 | `dialogue` | 有 | 是 | 人物交流、冲突、情绪爆点 |
| 旁白 | `narration` | 无 | 不限 | 推进剧情、时间跳跃 |
| 内心独白 | `internal_monologue` | 无 | 是 | 心理戏、情绪强化 |
| 画外音 | `voice_over` | 无 | 否 | 回忆、跨时空 |
| 现场画外音 | `off_screen` | 有 | 否 | 门外、电话另一端、画外角色 |
| 环境语音 | `ambient_voice` | 不重要 | 不重要 | 群众、广播、市场、走廊人声 |

在 prompt 中显式标注类型，模型更容易判断嘴型和声源位置。

## 长度规范

| 类型 | 推荐长度 |
| --- | --- |
| 单句对白 | 3-12 字 |
| 内心独白 | 1-2 句 |
| 旁白 | 1 句 |
| 连续对白 | 不超过 3 轮 |
| 单镜头总语音 | 不超过 8 秒 |

超过范围时，优先拆镜头、改成旁白 / 内心独白，或把信息落到视觉动作和道具。

## Dialogue 普通对白

用途：冲突、试探、告白、关系转折。对白要短，且必须能配合角色嘴型。

```text
冷调写实情绪风格，约8秒，16:9，雨夜屋顶，潮湿压抑

【分镜时序】
镜头1：two shot（双人镜头） static shot（固定镜头），角色A 和角色B 相隔两步站在屋顶边缘，角色A 肩背绷紧、角色B 眼神躲闪，雨水让两人之间的地面反光，雨声持续。
镜头2：medium close-up（中近景） over-the-shoulder（过肩），镜头从角色B 肩后看向角色A，角色A 抬眼、嘴角停住，霓虹光落在她脸侧，她低声说{你骗我}。
镜头3：close-up（近景） cut（切镜），角色B 沉默不语、眼神向旁边闪开，身后城市灯光模糊，雨声短暂变清楚。

【声音】
dialogue：角色A {你骗我}；雨声、远处车流声、衣料被雨打湿的细声。

【参考素材】
@图片1 作为角色A形象参考，@图片2 作为角色B形象参考，@图片3 雨夜屋顶参考
```

反例：长句会拖垮嘴型。

```text
Dialogue — 角色A：「我一直以为你是真心爱我的，可直到今天我才发现你从头到尾都在骗我。」
```

## Narration 旁白

用途：时间跳跃、人物处境、爽点信息。旁白只负责最关键的一句，剩下交给画面。

```text
高级商业写实风格，约10秒，16:9，酒店大堂冷暖光交织

【分镜时序】
镜头1：wide shot（远景） static shot（固定镜头），酒店大堂里所有人转头看向入口，门口光线把空间切开，大堂环境声轻微回荡。
镜头2：medium shot（中景） slow tracking（缓慢跟拍），角色A 从旋转门缓步入场，肩背挺直、步伐稳定，人群自动让出一条路，narration：{三年后，他回来了}。
镜头3：close-up（近景） cut（切镜），员工下意识起身、眼神惊愕，玻璃门反光在背景里滑过，椅子轻响。

【声音】
narration：{三年后，他回来了}；大堂环境声、脚步声、椅子轻响。
```

## Internal Monologue 内心独白

用途：心理戏、复仇、觉醒、克制情绪。没有嘴型压力，适合短句。

```text
冷峻电影感，约6秒，16:9，远观婚礼仪式，光影偏冷

【分镜时序】
镜头1：wide shot（远景） static shot（固定镜头），婚礼现场全景中宾客坐满主厅，角色A 站在远处廊柱阴影里，礼堂人声混响很远。
镜头2：close-up（近景） slow push in（缓慢推近），角色A 侧脸几乎不动、眼神锁定主桌，冷光切过眼尾，internal_monologue：{他们都会还的}。
镜头3：medium close-up（中近景） static shot（固定镜头），她缓慢松开攥紧的手指，婚礼灯光在瞳孔里晃动，远处掌声被压低。
```

反例：内心独白不要解释剧情。

```text
Internal Monologue — 角色A：「因为他当年背叛了我，所以我现在必须复仇。」
```

## Voice Over 画外音

用途：回忆、跨时空、多时空叙事。和旁白不同，画外音更像来自某个角色或记忆。

```text
怀旧胶片质感，约8秒，16:9，向日葵田，金色夕阳

【分镜时序】
镜头1：wide shot（远景） static shot（固定镜头），向日葵田铺满画面，小孩从画面右侧跑向远处，风吹花叶的声音清楚。
镜头2：medium shot（中景） tracking shot（跟拍），镜头跟随小孩穿过花海，小孩抬起手臂拨开花叶，逆光让衣角透亮，voice_over：{妈妈说过，人会消失两次}。
镜头3：close-up（近景） cut（切镜），小孩抬头望向天空，花影掠过脸侧，虫鸣和微风停在尾声。
```

## Off Screen 现场画外音

用途：房间外声音、电话另一端、门外喊话、未出场角色。要明确画外，避免模型给画内人物加嘴型。

```text
都市夜景写实风格，约6秒，9:16，昏暗走廊

【分镜时序】
镜头1：medium shot（中景） static shot（固定镜头），角色A 独自站在走廊尽头，右手慢慢攥紧钥匙，走廊灯光把空间拉长，冰箱低鸣从远处传来。
镜头2：close-up（近景） slow push in（缓慢推近），角色A 侧脸凝住、呼吸压低，门缝里的暖光落在肩上，off_screen：角色B（画外）{饭好了}。
```

## Ambient Voice 环境语音

用途：市场议论、广播声、群众声、远处人声。一般不写具体台词。

```text
ambient_voice：市场人群低声议论、摊主吆喝、远处车辆倒车提示音，不需要具体台词。
```

## 最稳声音组合

| 组合 | 用途 |
| --- | --- |
| 视觉动作 + narration | 时间跳跃、爽点信息 |
| close-up + internal_monologue | 心理戏、情绪落点 |
| action sound + environmental sound | 动作戏、广告、物件展示 |
| off_screen + reaction shot | 门外、电话、未出场角色 |
| ambient_voice + two shot | 市场、办公室、走廊、公共空间 |

## 多模态音频引用

```text
@音频1 作为对白音色参考
@音频2 作为环境声参考
@音频3 作为动作音效参考
```

只有用户明确提供音乐素材或要求音乐时，才写：

```text
@音频4 作为用户指定音乐参考
```

## JSON / DSL 建议

如果要把声音结构化，字段值保持人类可读，不绑定任何业务系统。

```json
{
  "scene": {
    "visual": "雨夜屋顶，潮湿压抑",
    "references": {
      "characters": [
        { "name": "角色A", "referenceImageRef": "@图片1" },
        { "name": "角色B", "referenceImageRef": "@图片2" }
      ],
      "scene": { "referenceImageRef": "@图片3" }
    },
    "voices": [
      {
        "type": "dialogue",
        "character": "角色A",
        "text": "你骗我"
      },
      {
        "type": "ambient_voice",
        "text": "远处人群低声议论"
      }
    ],
    "sounds": [
      "雨声",
      "远处车流",
      "衣料被雨打湿的细声"
    ]
  }
}
```

## 最终原则

写不下去时，先砍长对白，再加可见动作、道具信息和同步声音。
