# 正向 Few-shot

这些示例用于校准“本地落盘剧本项目”的完成形态。不要逐字复用设定；复用的是产物结构、信息密度和写法。

## Few-shot 1：复仇爽文短剧

### 用户需求

```text
创作一个 24 集女频短剧。女主被丈夫一家欺负，离婚夜曝光真实身份，主情绪是复仇爽，要本地落盘。
```

### `01-concept.md` 正向片段

```markdown
# 她在离婚夜封神

## 一句话故事
被豪门丈夫当成保姆羞辱三年的女人，在离婚宴上公开签字，却当场被全球顶级医疗集团董事会请回主持继任仪式。

## 观众承诺
这个故事让观众感到：被轻视的人终于拿回身份、尊严和选择权。

## 主情绪
复仇爽，辅以压抑后的公开打脸和身份反转。

## 幻想兑现
现实缺口是“付出不被看见、尊严被亲密关系剥夺”。幻想兑现是“我不是没人要，是你们不配知道我是谁”。

## 第一屏钩子
离婚宴上，婆婆把一枚硬币扔进红酒杯：“三年保姆费，拿着滚。”
```

### `02-characters.md` 正向片段

```markdown
## 林晚
- 外显身份：被豪门轻视的全职妻子。
- 内含欲望：拿回母亲留下的医疗集团，不再让任何人替她定义价值。
- 伤口或秘密：三年前为救丈夫隐瞒身份，也因此错过母亲临终。
- 可执行目标：今晚签字离婚，明早接任集团。
- 第一集弱势：在所有宾客面前被当成攀附豪门的笑话。
- 中后段反击能力：掌握丈夫公司命脉，能决定其融资生死。
- 情绪功能：让观众先替她憋屈，再看她把羞辱逐条还回去。

## 顾承泽
- 外显身份：自负的豪门继承人。
- 压迫手段：用婚姻、钱和公众羞辱控制林晚。
- 误判主角：以为林晚离开顾家就活不下去。
- 阶段性升级动作：冻结银行卡、抢走母亲遗物、公开宣布再婚。
- 被打脸代价：融资宴上被投资人当众撤资。
```

### `episodes/ep-001.md` 正向片段

````markdown
# 第 1 集：三年保姆费

## 本集目标
- 改变了什么：林晚从被羞辱的妻子，变成即将被看见真实身份的人。
- 主情绪：压抑、羞辱、反击前的冷静。
- 通向下一集的钩子：医疗集团董事会代表推门进场，叫她“林董”。

## 剧本正文

```fountain
INT. 酒店宴会厅 - 夜

婆婆把离婚协议推到林晚面前。

                    婆婆
          签吧。承泽肯给你三年体面，已经是顾家仁慈。

林晚低头看协议，没有立刻动笔。

顾承泽的新欢苏曼轻笑，把一杯红酒放到林晚手边。

                    苏曼
          姐姐，别舍不得。你这种人离开顾家，至少还能去做保洁。

宾客低声笑开。

婆婆从包里摸出一枚硬币，扔进红酒杯。

硬币沉底，撞出清脆一声。

                    婆婆
          三年保姆费，拿着滚。

林晚抬眼，第一次笑了。

                    林晚
          顾家算账，一直这么便宜？
```

## 卡点
宴会厅大门被推开，黑西装队伍进入。为首的老人停在林晚面前，躬身。

老人：“林董，董事会等您回去主持继任仪式。”
````

### `quality-check.md` 正向片段

```markdown
# 质量检查

## 通过项
- 开头 15 秒内出现羞辱动作：硬币扔进红酒杯。
- 观众承诺清晰：被轻视者拿回尊严。
- 主角没有靠解释反击，而是用身份和公共场合完成地位逆转。
- 集尾钩子强：所有人刚羞辱完她，马上听见“林董”。

## 当前最该修改的 3 个问题
1. 第 2 集需要立刻展示“林董”身份的现实权力，不能只停留在称呼。
2. 顾承泽的损失要具体化，例如融资、项目或家族继承权。
3. 苏曼不能只是嘴坏，要让她主动制造下一轮压力。
```

### `03-story-machine.md` 正向片段

```markdown
# 故事机器

## 主欲望
女主想在病情恶化前还清男主母亲的救命债，并让男主彻底恨她，好让他不被旧案牵连。

## 主阻碍
男主相信女主当年为钱背叛，反派持续伪造证据，让每次靠近都变成新的伤害。

## 核心信息差
- 观众知道：女主有苦衷，但不知道全部代价。
- 女主知道：当年她替男主认下事故责任，身体也因此受损。
- 男主知道：女主拿钱离开，但不知道钱被用来救他母亲。
- 反派知道：真相一旦曝光，自己会失去继承权。

## 预期循环
1. 男主以为自己掌控羞辱女主的局面。
2. 女主为了保护他，主动承受更坏误解。
3. 观众看见女主独自付代价，情绪转为心疼。
4. 集尾释放一小块证据，让男主的恨发生动摇。

## 升级阶梯
1. 公开重逢羞辱。
2. 合同绑定，女主不得不留在男主身边。
3. 旧物出现，男主误会加深。
4. 女主病情暴露一角。
5. 反派伪造女主二次背叛。
6. 男主发现母亲手术费来源。
7. 真相公开，但女主失踪。
```

### `episodes/ep-001.md` 正向片段

````markdown
# 第 1 集：她回来了

## 本集目标
- 改变了什么：男女主从断联旧爱，变成被合同重新捆绑的人。
- 主情绪：刺痛、误会、压抑心疼。
- 通向下一集的钩子：男主母亲认出女主声音。

## 剧本正文

```fountain
EXT. 医院门口 - 夜雨

女主攥紧缴费单。男主从黑色轿车旁回头。

                    男主
          缺钱了，才想起回来？

女主把缴费单折进掌心。

                    女主
          我不是来找你的。

男主把一份合同扔到她脚边。

                    男主
          签了。照顾我母亲一个月，钱给你。

女主弯腰捡合同，袖口滑开，露出一道旧伤。

男主抓住她手腕。

                    男主
          这伤怎么来的？

缴费单掉在地上，日期正是三年前事故当天。
```

## 结尾卡点
病房门口，一个熟悉的老妇声音传来：“知夏，是你吗？”
````

### `quality-check.md` 正向片段

```markdown
# 质量检查

## 通过项
- 虐点来自信息差，不是角色无理由发疯。
- 女主每次退让都有保护男主的具体原因。
- 男主的恨有证据支撑，后续反转才有重量。
- 第一集视觉焦点明确：雨、缴费单、合同、旧伤。

## 当前最该修改的 3 个问题
1. 男主不能只靠冷暴力推进，要给他可执行目标。
2. 反派需要尽早出手，否则误会来源会显得单薄。
3. 每集结尾应释放一个可追查证据，避免只有情绪重复。
```
