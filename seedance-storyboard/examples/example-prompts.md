# Seedance 分镜 Prompt 示例

以下示例使用 `镜头N` 时序化结构。每个镜头按“景别+运镜/切换 -> 主体动作与表情 -> 位置/空间变化 -> 同步声音”书写。示例默认不写音乐。

## 示例一：对白关系戏

场景：深夜训练室里，新兵向教官请求最后一次机会。

```text
冷调写实剧情风格，约10秒，16:9，深夜训练室的压抑氛围

【分镜时序】
镜头1：wide shot（远景） static shot（固定镜头），教官站得笔直、新兵停在数步之外并微微攥紧训练服下摆，空旷训练室只亮着冷白顶灯，两人被器械阴影隔开，远处空调低鸣压住空气。
镜头2：medium close-up（中近景） over-the-shoulder（过肩），镜头从教官肩后看向新兵，新兵抬眼又短暂停住、右手慢慢松开衣角，两人距离没有拉近，训练鞋轻轻摩擦地面的声音很清楚，新兵低声说{再给我一次机会}。
镜头3：close-up（近景） slow push in（缓慢推近），教官眼神轻微收紧、下颌线绷住，冷白灯在他脸侧形成硬阴影，空调声短暂停顿般变弱，他压低声音说{你最好配得上这句话}。

【声音】
dialogue：{再给我一次机会}、{你最好配得上这句话}；空调低鸣、鞋底摩擦声、训练室轻微混响。

【参考素材】
@图片1 作为新兵形象参考，@图片2 作为教官形象参考，@图片3 作为训练室空间参考
```

## 示例二：动作场面

场景：武侠竹林中两人短兵相接。

```text
冷色写实武侠风格，约10秒，16:9，薄雾竹林，空气湿冷，杀气压低

【分镜时序】
镜头1：wide shot（远景） static shot（固定镜头），黑衣刀客与持剑人相隔数步站定、肩背同时压低，竹林纵深被薄雾分层，风吹竹叶持续晃动。
镜头2：medium shot（中景） tracking shot（跟拍），黑衣刀客右脚用力蹬地向前冲、长刀从画面右侧斜切而入，持剑人侧身后撤并抬腕格挡，两人从雾气中段冲到前景，脚步踩碎竹叶声和刀剑碰撞声同时响起。
镜头3：close-up（近景） cut（切镜），两件兵器交错僵持，黑衣刀客眼神压低、持剑人手腕微微颤动，前景竹叶短暂掠过刀锋，金属余音在竹林里回荡。

【声音】
风声、竹叶摩擦声、脚步声、刀剑碰撞声、角色急促呼吸。

【参考素材】
@图片1 角色A造型参考，@图片2 角色B造型参考，@视频1 参考动作节奏
```

## 示例三：广告展示

场景：一支高端护肤精华广告。

```text
高端商业广告风格，约12秒，16:9，干净克制的暖白色空间，质感高级细腻

【分镜时序】
镜头1：close-up（近景） static shot（固定镜头），精华瓶口的透明液体沿玻璃边缘缓慢滑落，瓶身位于石材台面中央偏右，液滴轻碰玻璃的细小声音清晰。
镜头2：medium close-up（中近景） slow push in（缓慢推近），手指轻轻拧开滴管盖、动作平稳细腻，产品轮廓和金属反光逐渐成为画面中心，玻璃摩擦声很轻。
镜头3：medium shot（中景） pull out（缓慢拉开），产品静置在柔和漫射光中，台面、背景和瓶身形成完整展示关系，空间环境声干净克制。

【声音】
玻璃轻响、液滴声、手指拧盖声、安静室内环境声。

【参考素材】
@图片1 产品外观参考，@图片2 材质与包装细节参考
```

## 示例四：都市情绪戏

场景：雨夜便利店门口，女生隔着玻璃看见前任却没有进去。

```text
都市情绪电影感，约8秒，9:16，雨夜霓虹与便利店白光交织，潮湿、克制、略带失落

【分镜时序】
镜头1：medium shot（中景） static shot（固定镜头），女生停在便利店玻璃门外、肩膀微微收紧，雨滴和霓虹反光把门内外分成两层，雨声与便利店门铃声很轻。
镜头2：close-up（近景） slow push in（缓慢推近），她隔着玻璃看见前任坐在店内，眼神停住、呼吸在玻璃上留下浅雾，室内暖光和街边霓虹叠在她侧脸上，店内模糊人声从玻璃后传来。
镜头3：medium close-up（中近景） cut（切镜），她的手指靠近门把又慢慢收回，身体向后退半步离开门口，雨声重新盖过室内人声。

【声音】
雨声、便利店门铃、玻璃后的模糊人声、轻微呼吸声。

【参考素材】
@图片1 角色参考，@图片2 雨夜便利店场景参考
```

## 示例五：视频延长 / 续拍

场景：延长一段滑板少年在海边滑行的视频。

```text
将 @视频1 延长约8秒，延续原视频的金色海边治愈风格与少年滑行节奏

【分镜时序】
镜头1：medium shot（中景） continue shot（延续镜头），少年承接原视频最后的滑行动作继续向前，膝盖微微弯曲、上身保持平衡，夕阳海面在画面右侧延续，滑板轮压过带水痕地面的声音不断。
镜头2：medium close-up（中近景） tracking shot（跟拍），他微微侧身避开迎面跑来的小孩，手臂自然张开维持平衡，人物从画面中段滑向海边方向，小孩笑声从旁边掠过。
镜头3：wide shot（远景） pull out（缓慢拉开），少年重新加速滑向海边，衣角被海风吹起，海岸线和晚霞重新展开，海风和滑轮声自然收束。

【声音】
延续原视频的海风、滑轮摩擦地面声、远处小孩笑声。

【参考素材】
@视频1 原视频，@图片1 可选角色一致性参考
```

## 示例六：导演感转译

场景：用户说“想要一点 Wong Kar-wai 式的都市暧昧感”。

```text
都市情绪化碎片风格，约10秒，9:16，狭窄夜色街巷里潮湿霓虹轻微晕染，情绪暧昧又克制

【分镜时序】
镜头1：medium shot（中景） static shot（固定镜头），女生站在招牌下、手指轻轻搭在伞柄上，男生从画面深处缓慢靠近，两人被前景路人与雨伞短暂遮挡，雨声和远处车流声交错。
镜头2：close-up（近景） slow push in（缓慢推近），女生侧脸与玻璃中的男生轮廓短暂重叠，她的眼神停住但没有回头，霓虹反射在玻璃上轻微失焦，街边脚步声从身后经过。
镜头3：medium close-up（中近景） cut（切镜），男生停在她身后一步距离、没有开口，女生的肩膀微微放松又重新绷住，雨声短暂变得清晰。

【声音】
雨声、远处车流、街边脚步、人群低语；无对白。

【参考素材】
@图片1 角色A参考，@图片2 角色B参考，@图片3 夜巷霓虹场景参考
```

> 这里把“Wong Kar-wai 式”转译为前景遮挡、反射构图、霓虹晕染、慢节奏停顿和未完成的关系距离，而不是在 prompt 里机械堆导演名。

## 示例七：内心独白心理戏

场景：女主站在前夫婚礼远处的廊柱阴影里，决心复仇。

```text
冷峻电影感风格，约6秒，16:9，远观婚礼仪式，光影偏冷

【分镜时序】
镜头1：wide shot（远景） static shot（固定镜头），婚礼现场全景中宾客坐满主厅，女主站在远处廊柱阴影里几乎隐入背景，礼堂人声混响很远。
镜头2：close-up（近景） slow push in（缓慢推近），女主侧脸几乎不动、眼神锁定主桌，冷光切过她的眼尾，internal_monologue：{他们都会还的}。
镜头3：medium close-up（中近景） static shot（固定镜头），她缓慢松开攥紧的手指，婚礼灯光在瞳孔里晃动，礼堂掌声从远处传来又被压低。

【声音】
internal_monologue：{他们都会还的}；礼堂远处人声、掌声、衣料摩擦声。

【参考素材】
@图片1 作为女主形象参考，@图片2 婚礼仪式空间参考
```
