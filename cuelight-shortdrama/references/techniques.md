# 技法库

## 信息差玩法

### 悬念：观众知道得更多

用于制造焦虑。

例子：

- 观众看见酒里有毒。
- 主角以为这是礼物。
- 张力来自观众等待危险碰撞。

### 悬疑：观众和主角知道得一样多

用于制造参与感。

例子：

- 一个线索出现。
- 主角和观众一起解释它。
- 每个新线索都会改变假设。

### 底牌爽感：主角知道得更多

用于制造打脸和爽感。

例子：

- 反派羞辱主角。
- 观众知道主角有隐藏身份或隐藏能力。
- 爽点来自反派太晚意识到自己惹错了人。

## 信息牌投放表

把“谁知道什么”落成可拍的信息牌，不要只靠解释。

| 投放方式 | 适合投放的信息 | 使用提醒 |
| --- | --- | --- |
| 台词 | 关系立场、威胁、试探、误导、公开羞辱 | 不说世界观说明书；让台词带目的 |
| 动作/表情 | 隐忍、心虚、动摇、偏爱、恐惧、压迫感 | 只写可见动作，不写抽象心理 |
| 道具/环境 | 身份证据、旧伤、时间线、资源变化、秘密入口 | 道具要能改变后续行动，不做装饰 |
| 旁白/闪回 | 必要前史、复杂规则、观众必须先知道的情绪原因 | 短用；闪回必须反向照亮当前冲突 |

## 预期循环

用预期循环让故事运动可见：

1. 设置欲望：主角想要 X。
2. 设置阻碍：X 被人、系统、秘密或期限挡住。
3. 升级代价：失败会更痛。
4. 给出局部兑现或反转。
5. 打开下一个预期。

每个循环至少升级一个维度：

- 地位
- 资源
- 危险
- 亲密度
- 秘密价值
- 公开程度
- 道德代价

## 人物设计

把主角设计成观众的替身和载具。

最低可用人物包：

- **外显**：一眼能识别的标签，如落魄千金、隐藏战神、冷面总裁。
- **内含**：私人伤口、渴望、恐惧、柔软、羞耻或矛盾。
- **矛盾**：外显与内含之间的冲突点。
- **和谐机制**：为什么这个矛盾可信。

人物事件应该逼角色在“面具”和“真相”之间做选择。

### 人物 4 维到 8 要素

把四维继续转成八要素，避免人物只有标签没有行动。

| 四维来源 | 八要素 | 写法 |
| --- | --- | --- |
| 外显 | 身份 | 他在世界中的位置、资源、被如何对待 |
| 内含 | 欲望 | 他必须得到什么，得不到会损失什么 |
| 矛盾 | 问题 | 阻碍他行动的现实缺口或心理缺口 |
| 矛盾 | 负价值观 | 他一开始相信的错误生存法则 |
| 打破和谐 | 动作 | 他会采取什么主动行为 |
| 打破和谐 | 阻碍 | 谁或什么让动作付出代价 |
| 打破和谐 | 结果 | 动作之后世界发生什么变化 |
| 成长 | 正价值观 | 他最终要学会或证明的生存法则 |

## 反派设计

反派是压力机器。反派应当：

- 主动阻挡主角。
- 制造羞辱、危险或道德压力。
- 暴露主角的伤口或弱点。
- 在失败时给出强烈反应，让兑现可见。

短剧里，反派可以更极端、更直白，只要能提升情绪清晰度。

## 节奏规则

- 在观众最想看兑现的地方放慢。
- 加速处理交代、赶路、铺垫和重复信息。
- 在情绪高点阻断，不要等场景冷掉。
- 即使主兑现要延迟，也要每隔几个 beat 给一个小兑现。
- 用围观反应、公众证明和身份变化延长爽感。

## 常用钩子

- 巴掌、亲吻、车祸、逮捕、背叛、曝光即将发生。
- 隐藏身份即将揭露。
- 电话或目击者打断告白/解释。
- 反派发现了错误线索。
- 主角做出反常选择。
- 盟友看似背叛。
- 证据出现，但还没有被解释。
- 旧伤口变成新的现实危险。

## 事件有效性

一个事件只有在改变以下内容时才有价值：

- 关系
- 地位
- 危险
- 信息
- 资源
- 情绪
- 未来选择

如果一个 beat 只是在解释背景，就压缩它，或把它改成压力中的揭示。

## 动作台词画面

正文优先级：动作 > 画面 > 台词。

- 动作：让角色用选择推进剧情。剧情动作要自然隐形，不要像提纲；情绪动作只放在转折、爆发、崩溃、偏爱等关键节点。
- 画面：用位置、距离、道具、旁观者反应证明情绪兑现。
- 台词：只保留有攻击、防守、试探、隐瞒、请求、拒绝、承诺或背叛功能的句子。

台词三层过滤：

1. 删废话：删掉寒暄、重复信息、作者解释。
2. 加潜台词：角色口中说 A，真正想要 B。
3. 留指纹：不同角色的句长、词汇、攻击方式和回避方式要不同。
