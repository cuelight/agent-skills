电影剧本最核心的写法是：

> **用可拍摄的画面和对白，讲清楚：在哪里、什么时候、谁做了什么、谁说了什么。**

它不是小说，也不是分镜脚本。电影剧本首先是 **文学剧本 / Screenplay**，后续才会拆成分镜、镜头表、拍摄计划和 AIGC 生成段。

---

# 一、电影剧本的基本结构

标准电影剧本通常由这些元素组成：

```text
场景标题
动作描写
人物名
对白
括号提示
声音标记
转场
特殊格式：插入物、蒙太奇、闪回、字幕等
```

基本格式如下：

```text
外景. 县城车站 - 清晨

雾气压在低矮的站房上。几辆长途客车停在泥地里，车顶绑满麻袋和纸箱。

山风，22岁，穿灰色棉大衣，推着一辆飞鸽自行车从人群中挤出来。

                    山风
          今天得先摸清楚价格。

                    票贩子（O.S.）
          妹子，去省城啊？

山风转头。票贩子三十多岁，皮夹克，笑得很熟。

                    票贩子
          我这儿有票，不用排队。

                    山风
          不坐车。

票贩子一愣。

                    票贩子
          不坐车你来车站干啥？

山风看向几箱刚卸下来的搪瓷杯。箱子上印着“广州制造”。

                    山风
          看货从哪儿来。

切至：
```

# 二、场景标题怎么写

场景标题，也叫 **Slugline / Scene Heading**，用于标明地点、内外景和时间。

标准格式：

```text
内/外景. 地点 - 时间
```

常见写法：

```text
外景. 红色荒原 - 黄昏
内景. 废弃驿站 - 夜
内/外景. 行驶中的悬浮摩托 - 连续
外景. 县城车站 - 清晨
```

英文 Master Scene Format 对应：

```text
EXT. RED WASTELAND - DUSK
INT. ABANDONED INN - NIGHT
INT./EXT. MOVING HOVERBIKE - CONTINUOUS
```

常用时间词：

```text
日
夜
清晨
上午
午后
黄昏
深夜
连续
稍后
同时
闪回
```

注意：
只要 **地点、时间、内外景发生明显变化**，通常就应该新开一个场景标题。

# 三、动作描写怎么写

动作描写只写 **镜头能看到、听到的内容**。

推荐写法：

```text
锈红色荒原广袤无垠。暗紫色沙尘暴在远处翻涌。

亚莉娅驾驶悬浮摩托从沙丘之间高速掠过，车身拖出一道蓝白色光痕。
```

不推荐写法：

```text
亚莉娅心里很紧张，她觉得自己可能会死在这里。
```

更好的电影剧本写法：

```text
亚莉娅握紧车把，指节发白。她看了一眼后视镜，呼吸短促，却没有减速。
```

核心原则：

| 要求    | 说明                      |
| ----- | ----------------------- |
| 用现在时  | “走进、看见、停下”，不要写“走进了、看见了” |
| 写可拍内容 | 动作、表情、声音、环境、物体变化        |
| 少写心理  | 心理要转成行为和表情              |
| 段落短   | 一段通常 1–4 行              |
| 不写小说腔 | 少用大段抒情和抽象形容             |
| 不滥写镜头 | 文学剧本不需要每句都写“镜头推近”       |

---

# 四、人物第一次出现怎么写

角色第一次出现时，要简洁交代身份、年龄感、外貌或气质。

```text
亚莉娅，19岁，黑色短发，穿一件磨损的彩色防辐射服。她眼神警觉，动作比同龄人更快。
```

或者：

```text
山风，22岁，灰色棉大衣，推着飞鸽自行车，眼神机敏，像是随时在计算什么。
```

不要第一次出现就写太多设定。电影剧本里角色信息最好分散到行动和对白中体现。

---

# 五、对白怎么写

对白格式：

```text
                    人物名
          对白内容。
```

示例：

```text
                    雅典娜-7
          后方热源信号数量——三百以上。

                    亚莉娅
          不绕开。

                    雅典娜-7
          那是沙尘暴中心。

                    亚莉娅
          我知道。
```

对白要求：

| 要求      | 说明                |
| ------- | ----------------- |
| 口语化     | 像真人说话，不像说明书       |
| 有潜台词    | 不要什么都直说           |
| 不替作者讲设定 | 避免“如你所知，我们这个世界……” |
| 控制长度    | 大段对白要谨慎           |
| 区分角色声音  | 不同角色的说话节奏、用词要不同   |

# 六、括号提示怎么写

括号提示，也叫 **Parenthetical**，用于简短说明语气、动作或说话对象。

```text
                    亚莉娅
              （压低声音）
          你刚才说，三百以上？

                    雅典娜-7
              （停顿）
          现在是四百二十七。
```

常见括号提示：

```text
（笑）
（停顿）
（压低声音）
（不看她）
（对通讯器）
（冷静地）
（打断）
```

注意：括号提示要少用，不要写成长段心理分析。

不推荐：

```text
                    亚莉娅
              （她心里非常恐惧，但她强迫自己表现得很坚强，因为她知道现在不能退缩）
          继续。
```

更推荐：

```text
亚莉娅的手微微发抖。她把油门推到底。

                    亚莉娅
          继续。
```

# 七、O.S. / V.O. 怎么用

这是电影剧本里很重要的声音标记。

## 1. O.S. / 画外

表示角色在现场，但画面中暂时看不到。

```text
                    票贩子（O.S.）
          妹子，去省城啊？
```

中文可写：

```text
                    票贩子（画外）
          妹子，去省城啊？
```

适合：角色在场景里，只是镜头没拍到他。

---

## 2. V.O. / 旁白

表示旁白、内心独白、回忆声、电话另一端等非现场声音。

```text
                    山风（V.O.）
          那年冬天，我第一次知道，货比人跑得更快。
```

中文可写：

```text
                    山风（旁白）
          那年冬天，我第一次知道，货比人跑得更快。
```

或：

```text
                    亚莉娅（内心独白）
          不能停。停下就死。
```

---

# 八、转场怎么写

常见转场：

```text
切至：
淡入：
淡出。
叠化至：
猛切至：
匹配剪辑至：
```

英文对应：

```text
CUT TO:
FADE IN:
FADE OUT.
DISSOLVE TO:
SMASH CUT TO:
MATCH CUT TO:
```

现代剧本里不用每个场景都写 `切至：`。
只有当转场有特殊意义时才写。

# 九、特殊格式

## 1. 插入物 Insert

用于强调纸条、屏幕、照片、标签、手机信息等。

```text
插入：纸箱标签

“广州制造”
```

或者：

```text
插入：系统界面

错误代码：DAO-404
```

---

## 2. 字幕 / 屏幕文字

```text
字幕：1983年，北方某县城。
```

英文常写：

```text
SUPER: 1983, A COUNTY TOWN IN NORTHERN CHINA.
```

---

## 3. 蒙太奇 Montage

用于一组快速画面。

```text
蒙太奇：

— 山风在货亭前记价格。
— 她蹲在路边翻看小本子。
— 一辆货车驶过，车身印着“广州”。
— 山风抬头，眼神越来越亮。

蒙太奇结束。
```

---

## 4. 一组镜头 Series of Shots

比蒙太奇更偏连续动作。

```text
一组镜头：

A）亚莉娅拉下护目镜。
B）雅典娜-7调出热源图。
C）悬浮摩托引擎亮起蓝白色光。
D）远处沙丘后，纳米兽群跃出。

一组镜头结束。
```

---

## 5. 闪回 Flashback

```text
闪回：

外景. 村口 - 黄昏

十岁的山风站在雨里，看着父亲离开。

闪回结束。
```

也可以写：

```text
外景. 村口 - 黄昏 - 闪回
```

# 十、Master Scene Format 的排版要求

电影剧本常用 **Master Scene Format**，尤其是英文电影工业标准。

## 基础排版

| 项目   | 常见要求                          |
| ---- | ----------------------------- |
| 纸张   | 好莱坞标准多用 US Letter；中文环境可用 A4   |
| 字体   | Courier 12pt；中文可用等宽字体或宋体/仿宋体系 |
| 行距   | 单倍行距或固定 12pt                  |
| 页边距  | 左边距较大，方便装订；上下左右保持统一           |
| 页码   | 通常右上角                         |
| 页数估算 | 1 页剧本 ≈ 1 分钟成片                |

如果用 A4 中文模板，建议：

```text
纸张：A4
字体：Courier New / 等宽字体 / 宋体
字号：12pt 或小四
行距：固定值 12pt 或单倍
段前：0
段后：0

页边距：
上：2.5–3.4 cm
下：2.5–3.4 cm
左：3.5–3.8 cm
右：2.5 cm
```

如果想让 A4 更接近好莱坞 Letter 页容量，可以把上下边距设大一点。

# 十一、一页一分钟怎么理解

电影剧本常说：

```text
1 页 ≈ 1 分钟成片
```

这个不是严格数学公式，而是行业估算。

一部 120 分钟电影通常约：

```text
110–130 页剧本
```

常见结构：

```text
第一幕：1–30 页，约 30 分钟
第二幕上半：30–60 页，约 30 分钟
第二幕下半：60–90 页，约 30 分钟
第三幕：90–120 页，约 30 分钟
```

或者按 8 个 Sequence：

```text
Sequence 1：1–15 页
Sequence 2：15–30 页
Sequence 3：30–45 页
Sequence 4：45–60 页
Sequence 5：60–75 页
Sequence 6：75–90 页
Sequence 7：90–105 页
Sequence 8：105–120 页
```

在系统设计里，“一页一分钟”应该体现在：

```text
ScriptPage
Scene
Sequence
Act
Film
```

不应该直接放在 Shot 上。

# 十二、电影剧本和分镜脚本的区别

## 电影剧本

重点是故事、人物、场景、对白和动作。

```text
外景. 红色荒原 - 黄昏

悬浮摩托从沙丘之间高速掠过，远处的沙尘暴像一堵暗紫色墙壁压来。

                    雅典娜-7
          后方热源信号数量——三百以上。
```

## 分镜脚本

重点是镜头怎么拍。

```text
镜头 001
景别：远景
时长：5s
画面：红色荒原全景，悬浮摩托从左侧掠过。
运镜：tracking shot
声音：风沙声、摩托低频轰鸣

镜头 002
景别：中景
时长：5s
画面：雅典娜-7坐在后座，抬手扫描后方热源。
对白：后方热源信号数量——三百以上。
```

所以关系是：

```text
电影剧本
→ 分场
→ 分镜组
→ 镜头
→ 视频生成
```

# 十三、针对 AIGC 的推荐写法

如果是为了 CueLight / AIGC 电影生成，电影剧本要保留标准剧本格式，但最好额外做到：

```text
每场有明确场景标题
每段动作可被分镜化
每句对白有明确说话人
重要角色、场景、道具能被识别
动作连续性清楚
不要写无法视觉化的抽象心理
```

例如：

不适合 AIGC：

```text
亚莉娅感到命运正在吞噬她。
```

适合 AIGC：

```text
亚莉娅抬头。天空裂开一道黑红色的数据纹，像伤口一样横贯云层。她的左眼亮起淡蓝色光。
```

AIGC 友好的剧本应该让系统能抽取：

```text
角色：亚莉娅、雅典娜-7
场景：红色荒原
道具：悬浮摩托、发光护目镜
动作：驾驶、扫描、拉下护目镜、加速
情绪：紧张、决绝
声音：风沙、引擎、警报、对白
```

# 十四、完整中文电影剧本示例

```text
外景. 红色荒原 - 黄昏

锈红色荒原广袤无垠。暗紫色沙尘暴在远处翻涌，像一堵缓慢移动的墙。

悬浮摩托从沙丘之间高速掠过，车身下方喷出蓝白色光流。

亚莉娅伏低身体，双手紧握车把。她的护目镜挂在额头，脸上沾满细沙。

雅典娜-7坐在后座，右手抬起，掌心投射出一片半透明热源图。

                    雅典娜-7
          后方热源信号数量——三百以上。

亚莉娅没有回头。

                    亚莉娅
          距离？

                    雅典娜-7
          最近目标，四百米。速度还在增加。

沙丘后方，几道黑色粒子组成的兽形轮廓跃出。红色光学眼在尘雾中亮起。

亚莉娅拉下发光护目镜。镜片亮起绿色扫描线。

                    雅典娜-7
          建议规避东南象限沙尘带。

                    亚莉娅
          不绕。

雅典娜-7看向前方。沙尘暴中心闪烁着紫色电光。

                    雅典娜-7
          那是死亡区。

亚莉娅把油门推到底。悬浮摩托猛然加速，沙尘被气流撕开。

                    亚莉娅
          那就从死亡区过去。

切至：
```

# 十五、最终总结

电影剧本的具体要求可以压缩成这几条：

```text
1. 每场戏用「内/外景 + 地点 + 时间」开头。
2. 动作描写只写可被镜头拍到、可被声音听到的内容。
3. 角色第一次出现要简洁交代外貌、年龄感、气质或身份。
4. 对白用「人物名 + 对白」格式，人物名独立一行。
5. 括号提示只做简短语气或动作说明，不写长心理。
6. O.S. 表示现场画外声，V.O. 表示旁白、内心或非现场声音。
7. 转场、插入物、蒙太奇、闪回、字幕按标准格式单独标注。
8. 不写小说式心理，不滥写导演镜头指令。
9. 排版统一，通常 1 页 ≈ 1 分钟。
10. 对 AIGC 来说，剧本要方便拆成 Scene、Beat、VideoSegment 和 Shot。
```

对于 CueLight 的电影模式，最适合的内部链路是：

```text
ScriptBlock 剧本文本
→ ScriptPage 页数估时
→ Scene 场景
→ Beat 情节点
→ VideoSegment 分镜组，10–30 秒
→ Shot 镜头
→ GenerationTask 生成任务
→ Asset 视频结果
→ Timeline 成片
```
