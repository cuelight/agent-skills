# Seedance Storyboard Skill

将一句想法、一个剧情片段或一组参考素材，快速整理成适合 Seedance 2.0 使用的通用分镜 prompt。

这是一个通用外部 skill，不依赖 CueLight 项目、数据库、CLI 或任何业务落库字段。它只负责把输入组织成更专业、更稳定的 Seedance 分镜提示词。

## 当前定位

默认采用“首稿优先”的工作方式：

- 直接理解用户已有信息
- 按画幅选择景别和站位
- 用 `镜头1 / 镜头2 / ...` 组织事件顺序
- 每个镜头写清景别、运镜、主体动作、空间变化和同步声音
- 只在缺少关键事实时追问极少量问题

## Skill 能做什么

- 把模糊创意扩成 4-15 秒 Seedance prompt
- 处理横屏 / 竖屏景别选择
- 处理 `two shot`、`over-the-shoulder`、正反打、反应镜头
- 处理跨镜头和跨分镜组衔接
- 处理视频延长、视频编辑、动作复刻、广告展示和情绪戏
- 将“导演感”翻译成具体镜头语言，而不是简单堆名字

## 典型例子

### 原始想法

> 一个女孩在便利店外面看见前任，想进去又没进去，想要一点都市电影感。

### Prompt Draft

```text
都市情绪电影感，约8秒，9:16，雨夜便利店外，潮湿霓虹与室内白光叠在一起，克制、暧昧、略带失落

【分镜时序】
镜头1：medium shot（中景） static shot（固定镜头），女孩停在玻璃门外、肩膀微微收紧，雨滴和霓虹反光把门内外分成两层，雨声与便利店门铃声很轻。
镜头2：close-up（近景） slow push in（缓慢推近），她隔着玻璃看见前任坐在店内，眼神停住、呼吸在玻璃上留下浅雾，室内暖光和街边霓虹叠在她侧脸上，店内模糊人声从玻璃后传来。
镜头3：medium close-up（中近景） cut（切镜），她的手指靠近门把又慢慢收回，身体向后退半步离开门口，雨声重新盖过室内人声。

【声音】
雨声、便利店门铃、玻璃后的模糊人声、轻微呼吸声。

【参考素材】
@图片1 作为角色参考，@图片2 作为便利店夜景氛围参考
```

## 文件说明

- `SKILL.md`：主技能逻辑和生成规则
- `quick-reference.md`：镜头、画幅、声音和衔接速查
- `templates/storyboard-template.md`：常见场景模板
- `examples/example-prompts.md`：完整示例
- `references/dialogue-narration.md`：对白、旁白和同步声音分类

## 设计原则

- 镜头语言优先于空泛形容词
- `镜头N` 时序优先于精确秒数
- 景别选择服从画幅和剪辑动机
- 短时长控制镜头复杂度，单条最多 1-8 镜
- 默认不写 BGM、配乐、背景音乐、音乐氛围
- prompt 正文不写 `【约束条件】`
