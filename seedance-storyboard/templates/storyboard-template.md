# Seedance 分镜模板

模板用于快速选择镜头策略，不是固定套表。输出时仍要根据用户素材、画幅和时长调整。

## 模板一：叙事对白场景

适用：双人关系戏、问答、试探、争执、告白。

```text
[写实剧情风格]，约[#]秒，[16:9/9:16]，[空间氛围]

【分镜时序】
镜头1：two shot（双人镜头） static shot（固定镜头），[角色A] 与 [角色B] 保持 [距离/站位]，两人的表情分别呈现 [情绪A] 和 [情绪B]，[场景物件/光线] 明确两人关系，[环境声] 稳定铺底。
镜头2：medium close-up（中近景） over-the-shoulder（过肩），镜头从 [角色B] 肩后看向 [角色A]，[角色A] [手部/眼神/肩背动作] 并说{短对白}，两人距离没有立刻拉近，[动作音效或环境声] 轻微突出。
镜头3：close-up（近景） cut（切镜），[角色B] 眼神微变、嘴角停住或呼吸压低，[背景空间/前景遮挡] 让关系压力更强，[短暂停顿或环境声] 承接。

【声音】
dialogue：{短对白}；同步环境声和动作音效。

【参考素材】
@图片1 作为角色A形象参考，@图片2 作为角色B形象参考，@图片3 作为场景参考
```

## 模板二：情绪特写场景

适用：崩溃、犹豫、觉醒、发现异常、压抑爆发。

```text
[冷调/暖调情绪风格]，约[#]秒，[9:16/16:9]，[情绪基调]

【分镜时序】
镜头1：medium shot（中景） static shot（固定镜头），[角色] 独自停在 [空间] 中，肩背微微绷紧、手指轻碰 [物件]，[空间关系] 让人物显得被隔开，[环境声] 轻轻压住。
镜头2：medium close-up（中近景） slow push in（缓慢推近），[角色] 缓慢抬眼或低头，眼眶、呼吸、嘴角或指尖出现细微变化，[光线/雨影/反射] 落在脸侧，[动作音效] 变得清楚。
镜头3：close-up（近景） static shot（固定镜头），画面收在 [眼神/手部/关键物]，[角色] 从停顿自然过渡到 [小动作]，[近处空间细节] 稳定不乱，[一句内心独白或环境声] 收尾。

【声音】
internal_monologue：{一句内心独白}；或只保留环境声、呼吸声、物件轻响。

【参考素材】
@图片1 作为角色参考，@图片2 作为场景光影参考
```

## 模板三：动作场面

适用：打斗、追逐、舞蹈、快速移动、能量释放。

```text
[动作写实风格]，约[#]秒，16:9，[动作基调]

【分镜时序】
镜头1：wide shot（远景） tracking shot（跟拍），[主体] 从 [空间起点] 开始移动，腿部发力、肩背前倾，[场景障碍物/人群] 交代行动路线，[脚步声/风声] 建立速度。
镜头2：medium shot（中景） handheld（轻微手持），[主体] 完成 [闪避/转身/格挡/起跳]，手臂和躯干动作连贯不过度夸张，[空间位置] 从前景移动到中景，[碰撞声/衣料摩擦声] 清楚。
镜头3：close-up（近景） cut（切镜），[主体] 的眼神或关键动作落点被放大，[背景运动余波] 继续滑过画面，[喘息声/落地声] 收束动作结果。

【声音】
动作音效、呼吸、脚步、环境回声；用户明确要求音乐时才补音乐。

【参考素材】
@图片1 作为主体造型参考，@视频1 参考动作节奏和运镜衔接
```

## 模板四：广告展示场景

适用：产品、品牌、材质、食物、美妆、珠宝、科技产品。

```text
[商业广告风格]，约[#]秒，[16:9/9:16]，[光影与材质]

【分镜时序】
镜头1：close-up（近景） static shot（固定镜头），[产品局部] 在柔和光线下出现，材质纹理和边缘反光清晰，[产品位置] 稳定居中或偏一侧，[细微物件声] 干净。
镜头2：medium close-up（中近景） slow push in（缓慢推近），[手/使用者] 轻轻触碰或打开 [产品]，动作缓慢连贯，[产品从台面/手中] 进入主视觉，[玻璃声/液滴声/包装声] 同步。
镜头3：medium shot（中景） pull out（缓慢拉开），[产品] 与使用场景形成完整关系，[空间背景] 保持整洁不过度抢戏，[环境声] 轻微收尾。

【声音】
产品使用音效、材质细节声、空间环境声；音乐仅在用户明确要求时添加。

【参考素材】
@图片1 产品外观参考，@图片2 材质细节参考
```

## 模板五：都市情绪 / 夜景氛围场景

适用：孤独、暧昧、擦肩而过、回忆、城市夜晚。

```text
[都市情绪电影感]，约[#]秒，9:16，[雨夜/霓虹/玻璃反射]

【分镜时序】
镜头1：medium shot（中景） static shot（固定镜头），[角色] 停在 [街口/便利店/车站]，肩膀微微收紧、脚步停住，[玻璃/雨滴/霓虹] 把空间分成内外两层，[雨声或车流声] 明确。
镜头2：close-up（近景） slow push in（缓慢推近），[角色] 眼神停在 [对象/门把/远处人影] 上，手指从靠近自然过渡到收回，[反射或前景遮挡] 让关系不被说破，[呼吸声或门铃声] 轻轻出现。
镜头3：medium close-up（中近景） cut（切镜），[角色] 微微低头或侧身离开，空间距离重新拉开，[背景人声/雨声] 留住未说出口的情绪。

【声音】
雨声、车流、人群、门铃、呼吸或短对白；默认不写音乐。

【参考素材】
@图片1 角色参考，@图片2 场景参考
```

## 模板六：视频延长 / 续拍

适用：接着拍、补剧情、延长动作、延长情绪尾韵。

```text
将 @视频1 延长约[#]秒，保持原视频主体、光影、运动惯性和声音环境。

【分镜时序】
镜头1：medium shot（中景） continue shot（延续镜头），承接 @视频1 最后一帧的主体姿态，[主体] 从上一动作自然过渡到 [新动作]，[原空间方向] 保持一致，[原环境声] 延续。
镜头2：wide shot（远景） tracking shot（跟拍），镜头跟随 [主体] 向 [新位置] 移动，[空间关系] 逐渐展开，[脚步/风声/物件声] 与动作同步。
镜头3：medium close-up（中近景） pull out（缓慢拉开），[主体] 完成动作后 [表情/姿态变化]，[空间余韵] 保持原视频风格，[声音] 自然收束。

【声音】
延续原视频环境声和动作声；不要新增未要求的音乐。

【参考素材】
@视频1 原视频，@图片1 可选角色或场景一致性参考
```

## 模板七：视频编辑 / 剧情改写

适用：换角色、改动作、改表情、颠覆原剧情。

```text
基于 @视频1 编辑，保留原视频的 [场景/运镜/主体位置/节奏]，修改 [具体变化]。

【分镜时序】
镜头1：medium shot（中景） match cut（匹配切换），保留原视频开头主体位置，[主体] 的 [表情/动作] 按新设定发生变化，[空间关系] 不突兀改变，[原环境声] 保持。
镜头2：close-up（近景） cut（切镜），突出 [被修改的关键表情/物件/动作]，[前景或背景] 与原片一致，[动作音效] 与变化同步。
镜头3：medium shot（中景） static shot（固定镜头），新结果稳定呈现，[主体] 完成 [新动作]，[空间状态] 与编辑目标一致，[环境声] 收尾。

【声音】
保留原环境声；只在指定位置加入必要对白或动作音效。

【参考素材】
@视频1 原视频，@图片1 可选替换主体或风格参考
```

## 模板八：旁白驱动型片段

适用：反转揭示、时间跳跃、人物回归、内心转折。

```text
[写实电影感]，约[#]秒，[16:9/9:16]，[空间氛围]

【分镜时序】

镜头2：medium shot（中景） slow tracking（缓慢跟拍），[主体] 从 [入口/远处/阴影] 进入画面，肩背、步伐和眼神外化状态，[空间中其他人] 出现反应，narration：{一句短旁白}。
镜头3：close-up（近景） slow push in（缓慢推近），[主体] 在 [关键位置] 停住，表情变化克制但明确，[环境声] 短暂停顿后恢复。

【声音】
narration：{一句短旁白}；环境声和动作音效同步。

【参考素材】
@图片1 角色形象参考，@图片2 场景空间参考
```
